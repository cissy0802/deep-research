# BigCat Deep Research

Research reports built with multi-agent investigation and adversarial fact-checking.
Every load-bearing claim is challenged by 3 independent verifiers against primary sources.

- Site: https://cissy0802.github.io/deep-research/
- Rebuild pages from markdown sources: `python3 build.py`
- Add a new piece: drop `slug.{zh,en}.md` (plain & deep editions) into `sources/`, register it in `ARTICLES` + `INDEX_ENTRIES` in `build.py`, re-run.
